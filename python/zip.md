zip() 使用可迭代对象列表作为参数，可以将可迭代对象中的对应元素打包入一个元组中，然后返回由这些元组组成的列表。
语法：
zip([iterable, ...])
解压：zip(*iterable) ==> tuple
````
res = zip(range(10), range(100, 120, 2), range(200, 230, 3))
resl = list(res)
resl

>>>a = [1,2,3]
>>> b = [4,5,6]
>>> c = [4,5,6,7,8]
>>> zipped = zip(a,b)     # 打包为元组的列表
[(1, 4), (2, 5), (3, 6)]
>>> zip(a,c)              # 元素个数与最短的列表一致
[(1, 4), (2, 5), (3, 6)]
>>> zip(*zipped)          # 与 zip 相反，可理解为解压，返回二维矩阵式
[(1, 2, 3), (4, 5, 6)]
````
